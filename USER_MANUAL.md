# Temple CRM - User Manual (യൂസർ മാനുവൽ)
**Kozhimamparamb Pooram Cheruthuruthy Desam**

This manual explains how to use the Temple CRM to manage devotees, collections, and expenses efficiently.

---

## 1. Dashboard (ഡാഷ്ബോർഡ്)
The Dashboard provides a bird's-eye view of the temple's financial status.

- **Total Devotees (ആകെ ഭക്തർ):** Total number of registered devotees in the system.
- **Total Collected (പിരിച്ചെടുത്ത തുക):** Total amount received so far.
- **Total Pending (കുടിശ്ശിക):** Amount yet to be collected from devotees.
- **Collection Trend (വരവ് വിവരങ്ങൾ):** A graph showing monthly collection progress.
- **Recent Collections (സമീപകാല പിരിവുകൾ):** List of the most recent payments recorded.

---

## 2. Devotee Management (ഭക്തരുടെ വിവരങ്ങൾ)
Manage the list of devotees and their contact information.

- **Adding a Devotee (ഭക്തനെ ചേർക്കുക):** Use the '+' button or the 'Add Devotee' button in the Devotees section. Enter Name, Phone, and Address.
- **Searching (തിരയുക):** Use the search bar to find devotees by Name, Phone Number, or Address.
- **Editing (വിവരങ്ങൾ പുതുക്കുക):** Click on a devotee's name to view their profile and update details.
- **Deleting (ഡിലീറ്റ് ചെയ്യുക):** Use the delete icon to remove a record (Note: This cannot be undone).

---

## 3. Collections & Receipts (പിരിവും രസീതും)
Track spiritual contributions and generate receipts.

- **Pooram Tracker (പൂരം ട്രാക്കർ):** Dedicated tracker for Annual Pooram contributions. Enter Book No, Leaf No, and Amount.
- **Vilakku Tracker (വിളക്ക് ട്രാക്കർ):** Specifically for Ayyappan Vilakku contributions.
- **General Collection (ജനറൽ കളക്ഷൻ):** For donations and other pooja contributions.
- **Logging a Payment (പണം രേഖപ്പെടുത്തുക):** Go to the devotee's profile, click 'Add Receipt', select the event type, and enter the amount.
- **Printing Receipts (രസീത് പ്രിന്റ് ചെയ്യുക):** After saving a payment, click 'Print Receipt' to generate a digital copy that can be printed or saved as PDF.

---

## 4. Expenses (ചെലവുകൾ)
Keep track of temple expenditures.

- **Logging Expenses (ചെലവ് ചേർക്കുക):** Record payments made for temple maintenance, festivals, or other activities.
- **Categories (വിഭാഗങ്ങൾ):** Group expenses into categories like 'Maintenance', 'Festival', 'Pooja Items', etc.
- **Monthly Summary (പ്രതിമാസ ചിലവ്):** View total expenditure for each month.

---

## 5. Settings & Data Management (ക്രമീകരണങ്ങളും ഡാറ്റയും)
Ensure your data is safe and manage portal preferences.

- **Backup Data (ബാക്കപ്പ് എടുക്കുക):** Download your entire database as a JSON file for safety.
- **Excel Export/Import (എക്സൽ ഇമ്പോർട്ട്/എക്സ്പോർട്ട്):** Download devotee lists as Excel files or upload existing spreadsheets to bulk-add records.
- **Privacy Mode (പ്രൈവസി മോഡ്):** Mask sensitive financial data on the screen when showing the dashboard to others.
- **Language (ഭാഷ):** Switch between English and Malayalam from the settings.
- **Purge Data (എല്ലാം മായ്ക്കുക):** Permanently delete all records (DANGER: Use with extreme caution).

---

## 6. Keyboard Shortcuts (കീബോർഡ് ഷോർട്ട്കട്ടുകൾ)
Use these keys to navigate faster:

- `d`: Dashboard
- `v`: Devotees List
- `c`: Collections History
- `e`: Expenses
- `s`: Settings
- `n`: New Devotee (Add)
- `?`: Show Shortcut Help
- `Ctrl + K`: Open Command Palette

---

## 7. WhatsApp Reminders (വാട്സാപ്പ് സന്ദേശം)
Send payment reminders directly to devotees.
- In the Devotee Profile, if there is a pending balance, you will see a 'WhatsApp' icon.
- Clicking it will open WhatsApp with a pre-filled reminder message.

---

*Note: This system works offline but syncs with the cloud when internet is available.*
